Customize your site Easily change, add, or create new themes

Only change what you need

Kickstrap themes override the default Bootstrap theme. This means you don't have to define a style for every element. Kickstrap will fall back to Bootstrap by default.

Change Themes

Currently, Kickstrap works best using a less compiler in development. Don't worry. Many are free and are dead simple to use. Check out Compiling Kickstrap if you need help.

Once you download your LESS compiler, navigate to extras/themes/settings/themes.less and simply comment or uncomment the theme you want to use.

For example:

Warning: Make sure only one @import line is uncommented at any time, or one theme will override another. 
// SANDRA
@import "sandra/less/sandra.less";

// CERULEAN
// @import "cerulean/less/bootswatch.less";

The "Sandra" theme is uncommented before "@import" so this theme will be shown. "Cerulean", however, is not.

Description

To use Amelia, comment out this @import line in extras/themes/settings.less and be sure to re-comment any other themes

// AMELIA
@import "amelia/less/bootswatch.less";

// SOME OTHER THEME
// @import "othertheme/less/theme.less";

// See above how we commented out "Some Other Theme".
